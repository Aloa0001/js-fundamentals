JavaScript Operators

  • Arithmetic Operators
    • + - addition
    • - - subtraction
    • * - multiplication
    • / - division
    • % - modulus 9 % 2 => 1
    • ** - exponential 2 ** 3 => 8
    • 10 * true = 10
      10 / false=Infinity
    • 10 + '10' = 20
      10 + 'F15' = '10F15'
    • 10 + true = 11
      10 + false = 10
    • 10 + null = 10
      10 + undefined = NaN
  • JS Basic Operators
    • Assignment =
        Shorthand assignment notation
      • x *= x => x = x * x
        x += x => x = x + x
        ...
    • Increment / Decrement Operators
      • postfix notation - the value is returned, and then apply the operator
        • x++ => x = x + 1
        • x-- => x = x - 1
      • prefix notation - the operator is applied, and then returned the new value
        • --x
        • ++x
    • Coma operator -
      • allows programmers to evaluate multiple expressions, and return the result of the last expression
        (a = 7, b = 3, a = b * a, a + b) will return 24
