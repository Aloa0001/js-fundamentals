Introduction to JavaScript

  • What is it?
    • a programming language for web
    • usually is embedded in html code to make static Web pages more interactive or dynamic
    • can be written in a .js script file, or embedded inside .html files
    • JS is designed to be run by the browser when is attached to html documents being displayed
  • Where to use?
    • JS can be used as a client-side programming language, and as a server-side programming language(with node.js)
    • JS is also called Vanilla JavaScript, referring to plain JavaScript, without any additional libraries like jQuery
  • Pros os knowing JS
    • JS is at the top of the most popular technologies
    • JS is used by many frameworks and libraries built on Vanilla JavaScript, like Angular, React, Node, jQuery, ...
    • Js is one of the three core technologies of web design&development along HTML and CSS
  • What can JS do?
    • dynamic interaction
    • add business logic
    • connect to other applications across Internet
    • dynamically adding/removing/changing content to a web page
    • add animation
    • form validation
    • ...
  • Limitations
    • client-side JS does not allow read/write files on the disk, copy them, or execute programs
    • Same Origin Policy - JS from one page may not access other pages is they come from different sites, domain, protocol or port
    • JS is a singled threaded programming language - no multi-threading capabilities
    Obs: the limitations are mostly for user protection - preventing a webpage from accessing private information or harming user's data - these limitations does not exist for server-side js
