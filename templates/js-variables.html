JavaScript Variables

In JS, a variable doesn't have a data type, only its value has

JS uses Dynamically Typing - the data type is automatically assigned to the value of a variable based on the value that it is stored

Type Coercion - JS converts one datatype to another - operators and functions automatically convert the values given to them to appropriate types 

    
        let x = '10';
        console.log(typeof x); // -> string

        let y = 2;
        console.log(typeof y); // -> number

        console.log(x - y); // -> 8 - type coercion - JS understands that var x has to be converted to number due to operator '-'
        console.log(typeof (x - y)); // -> number

  • every value in JS has a data type
  • primitive type - 7 types
    • String - sequence of chars
    • Number - integers, floating point numbers, ...
      • number type contains always floating point numbers, even when storing integers
      • special number type values: infinity, - infinity (ex: when diving to zero), and NaN (ex: when a number is expected, but the value provided is not a number - dividing a number to a string)
    • Boolean - true or false
    • Undefined - a type of a var that it is not initialized yet - the value it is not assigned yet
    • Null - non existing, unknown, or empty variable
    • Bigint - large numbers that can not be stored in number type
    • Symbol - used to create unique identifiers for objects
  • object type

  • Declaring js-variables
    • let
      • allows an empty declaration of the Variable
      • allows the future change of the value/type that the variable is storing
      • allows variable re-initialization or variable mutation
    • const
      • can not change the value assign at initialization
      • can not declare an empty variable as cont - must initialize the value
    • var
      • similar to let keyword
      • can declare empty variables, initialize later, change the value/type of the stored value
